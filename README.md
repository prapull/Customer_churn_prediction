# Customer Churn Prediction

## Overview
This project predicts whether a telecom customer will churn using supervised machine learning classification models.

## Dataset
Telco Customer Churn Dataset (Kaggle)  
⚠️ Dataset is NOT included in this repository.

## Approach
- Data cleaning and preprocessing
- Encoding categorical features
- Train–test split
- Classification model training
- Model evaluation using Accuracy, F1-score, and ROC-AUC

## Results
- Accuracy: ~79%
- ROC-AUC: ~0.81

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab

## How to Run
1. Upload the dataset to Google Drive
2. Update dataset path inside the notebook
3. Run all cells
